{% raw %}WERELD.{% endraw %}
{% raw %}Op Figuur XXVII.{% endraw %}
{% raw %}W ie ftrooid zyn geld, en word verftoord^{% endraw %}
{% raw %}Als hy de gunft van waarfchouw hoord,{% endraw %}
{% raw %}Door Een , die achter hem komt treeden ?{% endraw %}
{% raw %}In tegendeel is zyn befcheid{% endraw %}
{% raw %}Een vriend'lyk woord van dankbaarheid,{% endraw %}
{% raw %}Na 't recht van goed verftand en reden.{% endraw %}
{% raw %}Maar zo der iemant vallen laat,{% endraw %}
{% raw %}De fpefi van een hooger graad,{% endraw %}
{% raw %}Als Goud en Zilver ooit kan weezen,{% endraw %}
{% raw %}De Woorden van 't gefcheurd gemoed,{% endraw %}
{% raw %}En dat men hem de waarfchouw doet,{% endraw %}
{% raw %}Zo ftaat zyn ongcnaa te vreezen.{% endraw %}
{% raw %}Zo dat ervarentheid bevind ,{% endraw %}
{% raw %}Dat elk zyn Geld-beurs meer bemind.{% endraw %}
{% raw %}Als 't groot belang van 't Zielen leven;{% endraw %}
{% raw %}Want immers word 'er zo gehoord ,{% endraw %}
{% raw %}Dat elk van ieder ydel woord,{% endraw %}
{% raw %}Eens reekenfchap zal moeten geeven.'{% endraw %}
{% raw %}Zo iemant dan zyn 's heeren geld,{% endraw %}
{% raw %}Op peene hem ter hand gefteld,{% endraw %}
