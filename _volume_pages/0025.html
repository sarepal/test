{% raw %}WERELD.{% endraw %}
{% raw %}Op Figuur IV.{% endraw %}
{% raw %}; ö W el'kom lieflyk aangezicht,{% endraw %}
{% raw %}Zo fchoon verklaard in 't helder licht,{% endraw %}
{% raw %}Van zulk een Monfter uitgekroopen,{% endraw %}
{% raw %}Dat quaad en zondig leeven hiet{% endraw %}
{% raw %}Een grouwel, die 't met aandacht ziet,{% endraw %}
{% raw %}Om voor te fchrikken en te loopen.{% endraw %}
{% raw %}Hoe fchoon is nieuw gebooren Deugd,{% endraw %}
{% raw %}Daar zich den Engel in verheugd,{% endraw %}
{% raw %}Om and're blydfchap t'overtrefFen!{% endraw %}
{% raw %}Die eertyds, by den quaaden gang{% endraw %}
{% raw %}In 't ftof ging kruipen, als een flang,{% endraw %}
{% raw %}Mag nu zyn vlugt tot God verheffen.{% endraw %}
{% raw %}'t Is koft'lyk wel, kruip zo maar voort,{% endraw %}
{% raw %}Uit d'oude huid , door d'enge poort,{% endraw %}
{% raw %}Tot lieflykheid en welbehagen,{% endraw %}
{% raw %}Van 't Heilig oog, en al de geen{% endraw %}
{% raw %}Die langs den weg der wysheid treên,{% endraw %}
{% raw %}Om 't befte erfdeel weg te draagen.{% endraw %}
{% raw %}Wat is 't een overgroote zaak,{% endraw %}
{% raw %}Te zyn een Engel, of een Draak !{% endraw %}
