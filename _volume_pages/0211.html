{% raw %}Op Figuur XLIX.{% endraw %}
{% raw %}JLVampzalig End en droevig heene vaaren,{% endraw %}
{% raw %}Als 't zelve Ding, dat u ontlaften zouw,{% endraw %}
{% raw %}Uw leeven diend tot jammerlyk bezwaaren,{% endraw %}
{% raw %}Tot wee, en ach ! vertwyfeling en rouw!{% endraw %}
{% raw %}Verkeerde Schat, daar 't hert in leid begraaven,{% endraw %}
{% raw %}Nu vaart gy heen, en laat uw knecht geen loon,{% endraw %}
{% raw %}Voor al den dienft, van 't vroeg' en laate flaaven,{% endraw %}
{% raw %}Hoe ftinkt uw deugd, hoe leelyk is uw fchoon!{% endraw %}
{% raw %}Daar draagt hy 't heen, dien flyter aller dingen :{% endraw %}
{% raw %}Geen hand zo fterk die zynen gang doet ftaan ,{% endraw %}
{% raw %}Of zyn Gezel met weerwil kan bedwingen,{% endraw %}
{% raw %}Wanneer hy komt, om hand aan 't werk te flaan.{% endraw %}
{% raw %}Noch vrind, noch maag, noch goude en zilvere fchyven,{% endraw %}
{% raw %}Noch Jeugd,noch Staat,noch Priefter,noch Doöoor,{% endraw %}
{% raw %}Kan deze Schat behouden , of doen blyven,{% endraw %}
{% raw %}DeTydenDood, die gaander ftout meêdoor.{% endraw %}
{% raw %}6 Aardfe menfch ! in uw gezonde dagen,{% endraw %}
{% raw %}Slaa hand aan't werk, nu is 't noch uwen tyd,{% endraw %}
{% raw %}Op dat gy 't geen, dat and're zich beklaagen,{% endraw %}
{% raw %}Door wys beraad, voorzichtig vlied en'myd.{% endraw %}
