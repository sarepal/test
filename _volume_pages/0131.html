{% raw %}WERELD. m{% endraw %}
{% raw %}Hoe is men dan zo koel van grond,{% endraw %}
{% raw %}Als of men 't niet ter deeg verftond,{% endraw %}
{% raw %}En dezen Raad niet kon geloove ?{% endraw %}
{% raw %}Daar anders 't Oor, het woord van Goud,{% endraw %}
{% raw %}Zo trouw'lyk opvat, fluit, en houd,{% endraw %}
{% raw %}En denkt: Dat voordeel hoord geen doove.{% endraw %}
{% raw %}รถ Menfch ! die u een Chriften roemd,{% endraw %}
{% raw %}En Chriftus uwen Meefter noemd,{% endraw %}
{% raw %}Die u dien Gulden Boom leerd zetten,{% endraw %}
{% raw %}In uw behaagelyken Hof,{% endraw %}
{% raw %}Van alle wereldlyke ftof,{% endraw %}
{% raw %}Leerd eens wel op uw herte letten:{% endraw %}
{% raw %}Of 't Hemelfe of Aardfe goed{% endraw %}
{% raw %}Het zwaarfte weegt in uw gemoed;{% endraw %}
{% raw %}Want zo de Vriend zyn vriend geloofde,{% endraw %}
{% raw %}En liet dien Boom doch ongeplant,{% endraw %}
{% raw %}Zo zag men hoe het klaar verftand,{% endraw %}
{% raw %}Door aardfe duifterheid verdoofde.{% endraw %}
