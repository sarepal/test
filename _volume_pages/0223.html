{% raw %}NAREDEN.{% endraw %}
{% raw %}Met al haar omgedraaide zaaken,{% endraw %}
{% raw %}Behoorden (voor een fcherpe fpyt){% endraw %}
{% raw %}De ftompe zinnen fcherp te maaken.{% endraw %}
{% raw %}Wat helpt de vlyt, die ieder doet?{% endraw %}
{% raw %}Wat helpt het draaven op de wegen,{% endraw %}
{% raw %}Die niet en gaan, na't Eeuwig Goed,{% endraw %}
{% raw %}Daar 't al en al aan leid gelegen!{% endraw %}
{% raw %}De Wereld zyne onwaardigheid,{% endraw %}
{% raw %}Zal ieders hert, die haar beminden ,{% endraw %}
{% raw %}Of nu, of dan, tot zynder tyd ,{% endraw %}
{% raw %}In 't Icheiden van de koop bevinden.{% endraw %}
{% raw %}Maar die ze noch by tyds bevind,{% endraw %}
{% raw %}En zet ze uit de Stoel des Herten,{% endraw %}
{% raw %}Op dat den Hemel ruimte wind,{% endraw %}
{% raw %}Die zal geen achter nazien fmerten.{% endraw %}
{% raw %}Het achter nazien geeft verdriet,{% endraw %}
{% raw %}Van fchoone dingen die ver.vloogen ,{% endraw %}
{% raw %}Maar 't bly voor uitzien zoet geniet,{% endraw %}
{% raw %}Na 't allerbefte en wys be-oogen.{% endraw %}
