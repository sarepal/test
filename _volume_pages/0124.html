{% raw %}ii4 De O N W A A R D I G E{% endraw %}
{% raw %}Terwyl hy voor de voet gaat weiden,{% endraw %}
{% raw %}En vind aan 't einde een diepe gracht,{% endraw %}
{% raw %}Tot wee en ach van alle beide.{% endraw %}
{% raw %}Daar ftort de Wagen met het Paard,{% endraw %}
{% raw %}En 't Eed'le Deel, dat zich liet trekken,{% endraw %}
{% raw %}Van deze Dierclyken aard,{% endraw %}
{% raw %}Dat alle Duiv'len daar meê gekken.{% endraw %}
{% raw %}Maar 't Eng'len-Deel, verheft zyn oog.{% endraw %}
{% raw %}Na 't wezen boven alle zinnen,{% endraw %}
{% raw %}En zend zyn zuchten naar om hoog{% endraw %}
{% raw %}In hoop van 't Eeuwig Heil te winnen:{% endraw %}
{% raw %}Trekt dat het fterkfte voor het wiel ,{% endraw %}
{% raw %}Zo moet het dier'lyk leven mede;{% endraw %}
{% raw %}Dat is: Het Lichaam volgt de Ziel,{% endraw %}
{% raw %}Op zyn gewiflê en vafte trede.{% endraw %}
{% raw %}Hy fleept het dan geweldig voort,{% endraw %}
{% raw %}En zo hy blyft geftaag volheiden,{% endraw %}
{% raw %}Zo brengt hy 't voor des Hemels Poort,{% endraw %}
{% raw %}Daar lyf en ziel jjefcheide werden.{% endraw %}
