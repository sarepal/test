{% raw %}NAREDEN.{% endraw %}
{% raw %}Die heeft het rechte Ding van waarden.{% endraw %}
{% raw %}Zo was de Wereld hem tot nut,{% endraw %}
{% raw %}(Dewyl hem God daar in liet komen){% endraw %}
{% raw %}En hy uit dezen heeft geput,{% endraw %}
{% raw %}Het rechte Erfdeel aller vroomen^{% endraw %}
{% raw %}De Deugd, die op den Hemel zag,{% endraw %}
{% raw %}Als een Waardy van Eeuwig blyven ,{% endraw %}
{% raw %}Waar voor de Aarden onder lag,{% endraw %}
{% raw %}Wiens Heil niet eind'loos kan beklyven.{% endraw %}
{% raw %}Zo draagd de Aarde goede Vrucht,{% endraw %}
{% raw %}Om Brood te maaijen en te kneeden ,{% endraw %}
{% raw %}Voor graage en wyze Hemel-zucht,{% endraw %}
{% raw %}Tot waardigheid van Eeuw'ge Vreden.{% endraw %}
{% raw %}Doch Aarde zelver is geen fpys.{% endraw %}
{% raw %}De Wereld, is geen Schat der Zielen *{% endraw %}
{% raw %}Maar teeld, uit haar, het Paradys,{% endraw %}
{% raw %}Met al, die Gods geboden hielen :{% endraw %}
{% raw %}' Zo doet de man van goed beleid,{% endraw %}
{% raw %}De kloeke Bouwman , op den lande,{% endraw %}
{% raw %}Die uit de Meft zyn Vrucht bereid,{% endraw %}
{% raw %}En krygt het honderdvoud in hande.{% endraw %}
{% raw %}O f Daar-{% endraw %}
