{% raw %}LA Mcft op eenen hoop gelegen,{% endraw %}
{% raw %}En geeft niet als een vieze lucht,{% endraw %}
{% raw %}Maar op den Akker is 't een zegen,{% endraw %}
{% raw %}En baard een aangenaame vrucht.{% endraw %}
{% raw %}Dat veele onwyzen dit bedachten,{% endraw %}
{% raw %}In opzicht van haar Hoopen geld,{% endraw %}
{% raw %}En die op armoeds akker bragten,{% endraw %}
{% raw %}Zo kreegen zy een vruchtbaar veld.{% endraw %}
{% raw %}Maar elk is in gefleepen zinnen,{% endraw %}
{% raw %}Niet even wys, en even gaauw,{% endraw %}
{% raw %}Om met het weinig veel te winnen,{% endraw %}
{% raw %}Onkunde vind men by het graauw.{% endraw %}
{% raw %}Graauw mogen zulke lui wel hieten ,{% endraw %}
{% raw %}In ongeleerde dommigheid,{% endraw %}
{% raw %}Die van haar meft geen vrugt genieten,{% endraw %}
{% raw %}Dewyl hy niet werd uitgebreid.{% endraw %}
{% raw %}Meft zyn de dingen van der aarde,{% endraw %}
{% raw %}Meft is het werelds geld en goed,{% endraw %}
{% raw %}By een gehouden zonder waarde,{% endraw %}
{% raw %}En zonder voordeel voor't Gemoed.{% endraw %}
